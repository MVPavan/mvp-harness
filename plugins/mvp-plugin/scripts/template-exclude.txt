# Files matching these NEVER ship into mvp-plugin/template/; they stay root-only.
# This keeps the shipped template a strict SUBSET of the coding-ritual root

# only useful in the harness-maintainer repo, and must not land in adopted repos.

# pattern covers every future addition without editing this file each time.
#
# Patterns are relative to each tree root (.claude/ or .codex/); a leading `/`
# anchors to that root, a trailing `/` matches a directory. build-template.sh
# feeds this file to rsync via --exclude-from.
